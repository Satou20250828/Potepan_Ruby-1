# メモアプリ（Ruby）

## 概要
プログラミングスクールの実践課題として、  
Rubyで作成したコンソール上で動作するメモアプリです。  
メモの追加や一覧表示といった基本機能を通して、  
ユーザー入力の受け取りやデータ管理の基礎を学びました。

## 作成した目的
本アプリは、Rubyの基礎文法を用いて  
データを保持・操作する処理の流れを理解することを目的として作成しました。  
じゃんけんアプリより一歩進み、配列やハッシュを使った  
情報管理に挑戦する課題として取り組みました。

## 使用技術
- Ruby  
- ターミナル（コマンドライン）

## 実装した機能
- メモの新規作成機能  
- メモの一覧表示機能  
- ユーザー入力を受け取る処理  
- 不正な入力に対するエラーメッセージ表示  

※ ファイル保存を実装した場合  
- メモ内容をファイルに保存  
- アプリ再起動後もメモを保持  

## 工夫した点
- ユーザーが操作に迷わないよう、選択肢を画面に表示しました。
- 入力内容が空の場合は登録できないようにし、  
  最低限の入力チェックを行いました。
- 処理の流れが分かりやすくなるよう、  
  コードを整理して記述することを意識しました。

## 苦労した点
メモをどのように管理すればよいか分からず、  
最初は実装に時間がかかりました。  
そこで「入力 → 保存 → 表示」という処理の流れを整理し、  
一つずつ動作を確認しながら実装を進めました。  

この経験から、  
コードを書く前に処理の流れを整理して考えることの重要性を学びました。

## 振り返り・今後の改善点
Rubyの基本文法を使って、  
簡単ではありますが実用的なアプリを作成する経験ができました。  
今後は以下のような改善を行い、理解を深めていきたいと考えています。

- メモの削除・編集機能の追加  
- メモを日付ごとに管理する機能  
- クラスを使った設計への改善
